# UAS-ALGORITMA-PEMROGRAMAN-LANJUT
Program pengolahan nilai mahasiswa berbasis bahasa C yang menggunakan array satu dimensi, array dua dimensi (matriks), perulangan, dan fungsi rekursif untuk mengelola data nilai mahasiswa dalam satu kelas. Program ini dibuat sebagai tugas Ujian Akhir Semester mata kuliah Algoritma dan Pemrograman Lanjutan.

# Deskripsi Program
Program ini merupakan aplikasi berbasis console yang dikembangkan menggunakan bahasa pemrograman C untuk mengelola dan mengolah data nilai mahasiswa dalam satu kelas. Program dirancang untuk membantu proses pengelolaan data akademik dengan memanfaatkan konsep array satu dimensi, array dua dimensi (matriks), perulangan, percabangan, serta fungsi rekursif.

Program memungkinkan pengguna untuk memasukkan, menampilkan, menambah, mengubah, dan menghapus data mahasiswa berdasarkan Nomor Induk Mahasiswa (NIM). Data nilai yang dikelola meliputi nilai Tugas, Ujian Tengah Semester (UTS), dan Ujian Akhir Semester (UAS). Selain itu, program juga melakukan pengolahan nilai berupa perhitungan rata-rata nilai setiap mahasiswa, rata-rata nilai kelas, serta penentuan mahasiswa dengan nilai tertinggi dan terendah.

# Repositori Isi
Repositori ini berisi file utama uas_alpro.c yang memuat seluruh source code program pengolahan nilai mahasiswa menggunakan bahasa C, termasuk pengelolaan data, pengolahan nilai, serta penerapan fungsi rekursif. Selain itu, repositori ini juga dilengkapi dengan file README.md yang berisi deskripsi program, struktur repositori, serta panduan cara mengompilasi dan menjalankan program.

# Cara Pengguna
Untuk menjalankan program, pengguna harus memastikan bahwa compiler bahasa C seperti GCC telah terpasang pada komputer. Setelah itu, buka folder tempat file program disimpan dan lakukan proses kompilasi menggunakan perintah gcc uas_alpro.c -o uas. Setelah proses kompilasi berhasil, program dapat dijalankan dengan perintah ./uas melalui terminal atau command prompt. Alternatif lain, program juga dapat dijalankan dengan membuka file uas_alpro.c menggunakan aplikasi Dev C++ atau Code::Blocks, kemudian memilih menu Compile & Run hingga program tampil pada layar console.
